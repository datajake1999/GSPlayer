1. Features
- PocketPC / PocketPC2002 / WM2003 / WM2003SE support
- MP3 playback using the MAD 0.15.1b
- Ogg Vorbis playback
- Wave format playback
- HTTP streaming playback (MP3 / Ogg)
- SHOUTcast streaming playback (MP3 / Ogg)
- ID3 v1/v1.1 support
- PLS/M3U playlist support
- 10 bands equalizer (MP3 playback only)
- Surround / Bassboost effects
- Reverb / Echo efffects
- Skin support
- Hardware button mapping 
- noiseless playback
- Sleep timer
- Hold (Lock hardware buttons)
- Decoder Plug-in interface


2. How to install
- Extract the setup files to your PC
- Execute setup.exe on your PC to install GSPlayer on your Pocket PC.
